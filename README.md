# Color Sort Puzzle 🎨

A fun color-sorting puzzle game built with React. Sort colored tiles into columns so each column contains only one color.

## Modes

- **🎨 Color** — standard color sorting
- **♦ Colorblind** — shapes/symbols instead of relying on color alone
- **🐾 Animals** — emoji animals with sound labels for each color

## Getting Started

```bash
npm install
npm run dev
```

## Deploy to Vercel

Push to GitHub, then import the repo on [vercel.com](https://vercel.com). Vercel auto-detects Vite — no config needed.

## Tech

- React 18
- Vite 5
- Zero external dependencies
